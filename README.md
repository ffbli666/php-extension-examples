# php-extension-examples
This project base on https://github.com/c9s/php-ext-skeleton
## swap
```sh
cd swap
phpize
./configure --enable-swap
make 
make test
make install
```
## mysort
```sh
cd mysort
phpize
./configure --enable-mysort
make 
make test
make install
```
